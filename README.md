# APEX_Grid_Fill_Series

**Overview**

Oracle APEX Interactive Grid does not natively support Excel-like drag-and-fill functionality. This enhancement introduces a way to replicate that behavior, allowing users to quickly copy and apply values across multiple rows.

**Objective**

To improve data entry efficiency by enabling users to drag and fill values in a manner similar to Excel.

**Approach**

The solution captures user interaction within grid cells and dynamically applies selected values across rows during a drag action, simulating Excel’s fill behavior.

**Benefits**
   - Faster data entry
   - Reduced manual effort
   - Improved user experience
   - Familiar Excel-like interaction

**Conclusion**

This enhancement brings a simple yet powerful usability improvement to Oracle APEX, making Interactive Grids more intuitive and efficient for end users.
